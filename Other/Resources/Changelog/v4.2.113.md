# Release notes

## Changes

- Added optional `env_vars: dict = None` parameter to all `run_program*` methods in `ProgramRunnerBase`, `ProgramRunnerPopen`, `ProgramRunnerSudo`, and `ScriptCollectionCore` — fully backwards compatible
- `ProgramRunnerPopen` merges the provided env vars on top of the current process environment before passing them to `Popen`
- `ProgramRunnerSudo` raises `ValueError` when `env_vars` is used (sudo env propagation not supported)
- `get_dotnet_build_diagnostics` now passes `DOTNET_CLI_UI_LANGUAGE=en-US` via the new `env_vars` parameter to force English output from dotnet build
